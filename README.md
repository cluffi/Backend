### HTTPS API для внутриигровых достижений

Минимальный HTTPS-сервер на C++ для получения внутриигровых достижений по HTTP-запросу.
Сервер использует WinSock2, OpenSSL (TLS) и SQLite и предназначен для учебных и исследовательских целей.

Проект не зависит от клиентской части: к API может обращаться любое приложение (игра, сайт, утилита) по стандартному HTTPS.

## Возможности
HTTPS-сервер (TLS поверх TCP)

Поддержка HTTP/1.1

Обработка GET-запросов

Получение параметров из строки запроса

Хранение достижений в SQLite

## Стек технологий
Язык: C++17

Платформа: Windows x64

Сеть: WinSock2

Шифрование: OpenSSL (TLS 1.2 / 1.3)

База данных: SQLite 3
